# v0.7 実用UI・Notion初期リスト

## UIの変更

- ホーム上部の大きなコピーと装飾を縮小
- 「作品を追加」「作品一覧」「検索」を先頭に配置
- 進行中の本・漫画・映画・アニメ・ドラマを同じカードで表示
- 一覧へ進捗と更新日を追加
- 選択中の作品は、全面朱色ではなく左線と薄い背景で表示
- フィルターを明るい操作パネルへ変更
- 詳細画面と入力画面を日常操作向けのサイズへ変更
- スマートフォンではカードを縦に並べる

## Notionから取り込むデータ

対象はNotionデータベース「📗 読書・コンテンツ」です。

初回は最新20件を取り込みます。作品名はAmazonなどの販売ページ名から読みやすい名称へ整えています。

保持する情報：

- 作品名
- 作者・制作者（確認できたもの）
- 作品種別
- 元URL
- NotionページURL
- Notion上の元タイトル
- Notionでの作成日・更新日
- ジャンル（確認できたもの）
- 殿堂入り、借りたい

`殿堂入り`は評価5などへ自動変換せず、タグとして保持します。

## 取り込み操作

v0.7をデプロイした後、ownerまたはadminでログインします。

```text
設定とデータ
→ Notionのリストを取り込む
→ Notionから取り込む
```

取り込み状況には、全20件のうち何件が登録済みか表示されます。

同じ操作を繰り返しても、`source_key`が一致する作品は自動でスキップされます。アプリで取り込んだ作品をソフト削除した場合は、後から再取り込みできます。

## 今後の追加

Notionの残りの項目は、同じ形式で次のバッチへ追加します。関連データの「ログ」「キーワード」「quote」「学び」は、作品本体の移行後にメモ・分類へ対応付けます。
