# 実装レポート v0.4

## 実装範囲

- Cloudflare Access JWTとD1の利用状態を組み合わせた二段階判定
- owner / admin / member / viewer
- 招待、一時停止、ブロック、解除、任意のAccessセッション失効
- 作品・体験・メモ・ジャンル・テーマ・タグ
- 再読・再視聴の履歴追加
- タイトル・作者・分類・メモの横断検索
- 読書を優先したホーム
- スマホ1カラム・下部ナビ、PC 3カラム
- JSON / CSV / Markdownエクスポート

## v0.2 入力・編集UX

- 媒体別の作者欄・状態名・進捗単位
- PC詳細ペインのクイック編集
- スマホの保存操作固定
- 下書き保存・復元・破棄
- 未保存離脱確認
- 進捗整合性の画面・API二重検証

## v0.3 検索・保存ビュー

- 複数語をすべて含むAND検索
- 分類名の部分一致と複数分類AND検索
- 検索結果の総件数をAPIで返却
- ジャンル・テーマ・タグ候補を使用回数順で表示
- 現在の検索・絞り込み条件を名前付き保存
- 保存ビューの適用、既定表示、削除
- 絞り込みチップを個別解除
- 保存ビューは利用者ごとに分離

## v0.4 メモ・体験履歴

- メモの編集・削除
- メモの手動並び替え
- 更新順、作成順、メモ種別順の表示切替
- 体験履歴の編集・削除
- 初回順、最新順の表示切替
- 前回評価との差分表示
- 初回評価から最新評価までの変化要約
- メモ・体験履歴の楽観ロック
- 履歴変更時の作品状態・評価・進捗再同期
- 体験メモと自動作成された印象メモの整合維持
- すべての編集・削除・並び替えを監査ログへ記録
